## Frame 11: Sync Architecture Diagram

**Positie:** 6100, 10100 | **Grootte:** 5900 x 2000px

### Mermaid Diagram: Sync Sequence

```mermaid
sequenceDiagram
    participant U as User
    participant GV as Gantt View
    participant PM as ProjectModel
    participant API as API Layer
    participant DB as Database
    participant CV as Calendar View
    participant TV as TaskBoard View

    U->>GV: Drag task (change date)
    GV->>PM: updateTask(id, newDates)
    PM->>PM: Validate changes
    PM->>API: PATCH /tasks/{id}
    API->>DB: UPDATE task SET...
    DB-->>API: Success
    API-->>PM: Updated task
    PM->>PM: Update stores
    PM-->>GV: Refresh view
    PM-->>CV: Broadcast update
    PM-->>TV: Broadcast update
    CV->>CV: Re-render event
    TV->>TV: Re-render card

    Note over GV,TV: Sync latency < 2 seconds (KR2.71)
```
